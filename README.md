**AnimeWatch** is a simple desktop program built with **Tkinter** that helps users manage the anime they watch.

## Features Implemented
- Add new anime to the watchlist  
- Remove anime from the watchlist  
- Mark episodes as watched  
- Full user interaction through  buttons and input   
- Data updates instantly (no refresh is needed!)
